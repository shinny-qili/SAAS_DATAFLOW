## 项目目录总览

- `0_按场景使用描述SaaS版/`：按业务场景梳理 SaaS 能力，包含交易员、交易指令、套保闭环等场景文档，用于描述需求背景与价值。
- `1_按子产品描述SaaS版/`：以子产品（合同宝、匹配宝等）为维度的功能说明，便于拆分模块化交付。
- `2_V1版系统总述/`：聚焦 V1 版本整体设计与增补说明，用来对外或对内快速了解首版范围。
- `3_V1版系统实现步骤/`：V1 实施落地的详细拆解，涵盖界面方案、配置流程、各业务模块的实现细节与图示。
- `assistant_outputs/`：组件库演示及导出产物，当前主要包含 `inerface/erp-app.html` 及其依赖（`vendor/`），用于静态展示。
- `workdir/`：本地迭代原型与临时脚本的工作目录，包含 `prototype.html`、`index.html` 等用于快速验证界面交互的文件。
- `产品宣传和网站方案.md`：面向外部宣传/官网内容的思路整理，可用于市场物料或官网信息的准备。

> 若需贡献新的文档或原型，请根据内容选择相应一级目录，保持命名规范（数字前缀 + 描述），便于后续查阅与版本管理。
