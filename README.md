## 项目目录总览

### 📌 重点文件（核心交付物）

以下文件是本项目的核心交付物，可直接用于产品演示、开发参考和用户培训：

1. **`workdir/prototype.html`** - **期现业务管理平台原型系统**
   - **作用**：完整的业务功能原型演示，包含所有核心模块的界面实现
   - **内容**：合同宝、指令宝、匹配宝、交易宝、风控宝、报表、全局设置等完整业务流程
   - **用途**：产品演示、开发参考、交互验证、用户培训
   - **特点**：单文件完整系统，包含所有业务逻辑和交互效果

2. **`assistant_outputs/inerface/erp-app.html`** - **ERP 组件库演示**
   - **作用**：Ant Design 组件库的完整演示和模板示例
   - **内容**：布局组件、输入组件、数据展示组件、业务组件、图表组件等
   - **用途**：组件库参考、UI 设计规范、开发模板
   - **特点**：包含组件代码示例、配置说明、使用场景

3. **`workdir/期现系统使用说明书.md`** - **系统使用说明书**
   - **作用**：面向最终用户的系统使用指南
   - **内容**：快速开始、功能模块说明、常用操作指南、常见问题解答
   - **用途**：用户培训、操作手册、产品交付文档
   - **特点**：与原型系统和产品方案文档保持一致，内容完整清晰

4. **`3_V1版系统实现步骤/cursor整理的方案-定稿版/`** - **产品方案文档（定稿版）**
   - **作用**：V1 版本的完整产品方案文档，作为开发实施的依据
   - **内容**：
     - `0_全局设置.md` - 全局设置模块产品方案
     - `1_合同宝.md` - 合同宝模块产品方案
     - `2_匹配宝与委托备注.md` - 匹配宝模块产品方案
     - `3_指令宝.md` - 指令宝模块产品方案
     - `4_交易宝.md` - 交易宝模块产品方案（含手机分辨率适配）
     - `7_风控宝.md` - 风控宝模块产品方案
     - `8_报表.md` - 报表模块产品方案
     - `9_个人工作台.md` - 个人工作台产品方案
     - `0_数据的单元和集成测试.md` - 测试方案文档
     - `0_系统缺省默认.md` - 系统默认配置说明
   - **用途**：开发实施依据、产品需求文档、技术方案参考
   - **特点**：内容详细完整，包含界面功能、数据来源、计算公式、数据约束、手机适配等

5. **`对外演示/roadshow.html`** - **产品路演展示页面**
   - **作用**：面向外部客户的产品路演和演示页面
   - **内容**：快期产业服务中心介绍、产品功能展示
   - **用途**：客户演示、产品推广、市场宣传
   - **特点**：简洁美观的展示页面，突出产品价值

### 📚 其他文件夹（迭代过程文档）

以下文件夹记录了产品方案的迭代过程，可作为字典配合 Cursor 查询某个设计的起因、经过、结果：

- `0_按场景使用描述SaaS版/`：按业务场景梳理 SaaS 能力，包含交易员、交易指令、套保闭环等场景文档，用于描述需求背景与价值。
- `1_按子产品描述SaaS版/`：以子产品（合同宝、匹配宝等）为维度的功能说明，便于拆分模块化交付。
- `2_V1版系统总述/`：聚焦 V1 版本整体设计与增补说明，用来对外或对内快速了解首版范围。
- `3_V1版系统实现步骤/`：V1 实施落地的详细拆解，涵盖界面方案、配置流程、各业务模块的实现细节与图示。
  - **注意**：`cursor整理的方案-定稿版/` 是该文件夹下的定稿版本，是最终的产品方案文档。
- `assistant_outputs/`：组件库演示及导出产物，当前主要包含 `inerface/erp-app.html` 及其依赖（`vendor/`），用于静态展示。
- `workdir/`：本地迭代原型与临时脚本的工作目录，包含 `prototype.html`、`index.html`、`期现系统使用说明书.md` 等用于快速验证界面交互的文件。
- `产品宣传和网站方案.md`：面向外部宣传/官网内容的思路整理，可用于市场物料或官网信息的准备。

### 🚀 快速开始

1. **查看原型系统**：打开 `workdir/prototype.html` 或通过 `index.html` 访问
2. **查看组件库**：打开 `assistant_outputs/inerface/erp-app.html`
3. **查看产品方案**：阅读 `3_V1版系统实现步骤/cursor整理的方案-定稿版/` 下的文档
4. **查看使用说明**：阅读 `workdir/期现系统使用说明书.md`
5. **查看路演页面**：打开 `对外演示/roadshow.html` 或通过 `index.html` 访问

> 若需贡献新的文档或原型，请根据内容选择相应一级目录，保持命名规范（数字前缀 + 描述），便于后续查阅与版本管理。
