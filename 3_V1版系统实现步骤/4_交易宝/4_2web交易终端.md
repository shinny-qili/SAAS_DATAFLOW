目录路径：
    一级：交易宝
    二级：Web交易终端

定位与终端形态
    浏览器运行，默认嵌入公司统一工作台，也可以独立窗口。
    目标体验靠近快期2，支持键盘快捷键、深色/浅色主题、布局自定义。
    通过HTTPS + WebSocket连接期现交易服务，支持断线提示与自动重连。

界面结构（默认布局）
    顶部状态条：
        交易账户下拉 + 登录状态
        行情连接状态指示灯，交易状态指示灯
        风险度与可用资金概览
        指令执行入口（跳转至侧滑抽屉）
    左侧行情与自选区域：
        自选列表123，自选合约维护
        五档盘口、最新价、涨跌幅、持仓量、成交量
        
        报价面板：自组合报价、增删改按钮
    中部下单板：
        自动开平下单板
            保值方案选择
            （三键：买开、平仓、卖开）
            支持设置数量快捷键（1/5/10手）、跟随行情价、滑点保护
        传统下单板
            字段：合约、方向、价格类型（限价/对价/市价）、开平、投保标志、交易所移仓标记、数量、有效期
            高级：自动开平开关、报单触发条件（立即/开盘/条件单引用）
        批量/组合下单面板
            双开/双平、移仓换月模板，可保存模板
    右侧数据面板（可切tab）：
        持仓表：方向、合约、手数、均价、浮盈亏、保值方案、投保标记
        委托表：状态、报单时间、触发方式（手动/自动）、关联指令
        成交表：价格、数量、手续费、撮合编号
        资金表：权益、可用、占用、风险度、保证金

交互与能力
    账户切换：切换即触发重新拉取账户快照与订阅，保留当前布局。
    保值方案切换：实时重新筛选持仓/委托，默认联动下单板的hedge_scheme_id。
    行情订阅：前端通过WebSocket发送订阅/退订消息，后端推送增量。
    快捷键：
        F1/F2/F3：三键下单
        Ctrl+数字：切换自选组
        Shift+Enter：快速提交委托
    多窗口：允许把任何表格弹出为浮层，适配多屏。
    模板：
        下单模板：保存合约、数量、投保、自动开平配置。

指令执行侧滑抽屉
    列表字段：指令类型、来源（人员/客商）、合同、合约、方向、数量、执行方式（人工/自动）、进度。
    操作：
        接收/拒绝
        一键按指令生成下单参数并回写执行结果
        自动执行时展示机器人状态与日志

风控与提示
    风险度超过阈值，高亮顶部指示，并禁止继续开仓，提示联系风控。
    委托价格偏离（超价）时弹Toast提醒。
    柜台/行情断线时，锁定下单按钮，仅保留“撤单”。

数据同步
    WebSocket推送：行情、委托、成交、连接状态。
    HTTP轮询：列表初始加载、历史查询（分页）。
    本地存储：保存布局、主题、快捷键设置。

集成接口（V1）
    GET /api/trade/web/settings            → 加载布局、快捷键、模板
    GET /api/trade/accounts                → 可用账户+状态
    GET /api/trade/hedge-schemes           → 当前交易员支持的保值方案
    POST /api/trade/orders                 → 下单
    POST /api/trade/orders/{id}/cancel     → 撤单
    GET /api/trade/orders                  → 委托列表
    GET /api/trade/fills                   → 成交列表
    GET /api/trade/positions               → 持仓列表
    GET /api/trade/closed                  → 平仓列表
    GET /api/trade/funds                   → 资金信息

测试要点
    账户切换后1s内完成行情与数据刷新。
    自动开平下单板在键盘模式下不能触发重复提交。
    指令执行抽屉的回写结果与交易服务事件一致。
